# Contributing to Shady SDK

We welcome contributions to improve privacy tooling for Solana.

## How to Contribute
1. Fork the repo and create a new branch.
2. Implement or fix features.
3. Run `pnpm -r build` and ensure no errors.
4. Submit a PR with a clear description.

## Code Style
- TypeScript ESNext
- Keep functions pure where possible
- All crypto logic must remain client-side
